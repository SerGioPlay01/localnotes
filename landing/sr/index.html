Прича о настанку

Зашто је настао Local Notes

Све је почело простим разочарењем — пронаћи апликацију за белешке која је заиста приватна, брза и бесплатна.

Проблеми постојећих апликација
☁️
Све се синхронизује са сервером. Notion, Evernote, Google Keep — ваше белешке живе на туђем рачунару. Један пробој и подаци су откривени.
💳
Шифровање кошта. Већина апликација крије право шифровање иза плаћеног плана. Приватност не би требало да буде премиум функција.
📋
Регистрација свуда. Да бисте написали белешку, потребни су вам имејл, лозинка и сагласност са политиком која дозвољава коришћење ваших података.
🌐
Нема офлајн подршке. Већина веб апликација је бескорисна без интернета. Ваше белешке треба да буду доступне увек и свуда.
Почетак потраге
Тражење приватне апликације за белешке
Испробане десетине апликација. Свака је захтевала налог, синхронизовала се са обласком или наплаћивала основне функције. Ништа се није чинило заиста приватним.
Спознаја
Прегледач већ има све потребно
Модерни прегледачи имају IndexedDB за локално складиштење, Web Crypto API за AES-256 и Service Workers за офлајн рад. Сервер није потребан.
Први прототип
Направљена минимална радна верзија
Једноставан едитор, локално складиштење, основно шифровање. Без фрејмворка, без зависности. Само HTML, CSS и чист JavaScript.
Развој идеје
Ознаке, календар, 12 језика, PWA
Додато форматирање текста, ознаке у боји, уграђени календар, претрага са транслитерацијом и потпуна локализација на 12 језика. Упаковано као PWA.

Формат .note

Ваши подаци, ваш формат

Датотека .note је преносиви шифровани контејнер дизајниран с једним циљем: ваше белешке припадају вама — не платформи, облаку ни компанији.

🔒
Рођен из проблема приватности
Свака популарна апликација за белешке чува податке на удаљеним серверима. Један пробој, један захтев власти, једна промена политике — и ваше приватне мисли су откривене. Формат .note то чини немогућим по својој природи.
📦
Шта је унутар .note датотеке
Датотека .note је самостални JSON пакет: садржај белешке, метаподаци, ознаке и — при укљученом шифровању — AES-256 шифровани текст који само ваша лозинка може дешифровати. Без власничког формата, без везивања за добављача.
🌐
Офлајн-прво, увек преносиво
Извезите било коју белешку као датотеку .note и носите је на USB-у, шаљите преко Сигнала или архивирајте на деценије. Увезите назад у Local Notes на било ком уређају — без налога, без интернета.
🛡️
Zero-knowledge као основа
При шифровању белешке кључ никада не напушта ваш уређај. Local Notes користи уграђени Web Crypto API прегледача — исти стандард који користе банке. Чак и ако неко пресретне вашу датотеку .note, видеће само шифровани текст.

Честа питања

Да, потпуно. Local Notes је open source софтвер под MIT лиценцом. Без плаћених планова, без скривених накнада — заувек. Све функције су доступне бесплатно.

Ваше белешке се чувају у IndexedDB вашег прегледача — локалној бази података уграђеној у сваки модерни прегледач. Ништа се никада не шаље на сервер.

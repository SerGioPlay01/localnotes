Priča o nastanku

Zašto je nastao Local Notes

Sve je počelo jednostavnom frustracijom — pronaći aplikaciju za bilješke koja je zaista privatna, brza i besplatna.

Problemi postojećih aplikacija
☁️
Sve se sinkronizira s poslužiteljem. Notion, Evernote, Google Keep — vaše bilješke žive na tuđem računalu. Jedan propust i podaci su otkriveni.
💳
Enkripcija košta. Većina aplikacija skriva pravu enkripciju iza plaćenog plana. Privatnost ne bi trebala biti premium značajka.
📋
Registracija svugdje. Da biste napisali bilješku, trebate email, lozinku i pristanak na pravila koja dopuštaju korištenje vaših podataka.
🌐
Nema podrške za offline. Većina web aplikacija je beskorisna bez interneta. Vaše bilješke trebaju biti dostupne uvijek i svugdje.
Početak potrage
Traženje privatne aplikacije za bilješke
Isprobane desetine aplikacija. Svaka je zahtijevala račun, sinkronizirala se s oblakom ili naplaćivala osnovne funkcije. Ništa se nije činilo zaista privatnim.
Spoznaja
Preglednik već ima sve što je potrebno
Moderni preglednici imaju IndexedDB za lokalno pohranjivanje, Web Crypto API za AES-256 i Service Workers za offline rad. Poslužitelj nije potreban.
Prvi prototip
Izrađena minimalna radna verzija
Jednostavan editor, lokalna pohrana, osnovna enkripcija. Bez frameworka, bez ovisnosti. Samo HTML, CSS i čisti JavaScript.
Razvoj ideje
Oznake, kalendar, 12 jezika, PWA
Dodano formatiranje teksta, oznake u boji, ugrađeni kalendar, pretraživanje s transliteracijom i potpuna lokalizacija na 12 jezika. Zapakirano kao PWA.

Format .note

Vaši podaci, vaš format

Datoteka .note je prenosivi šifrirani kontejner dizajniran s jednim ciljem: vaše bilješke pripadaju vama — ne platformi, oblaku ni tvrtki.

🔒
Rođen iz problema privatnosti
Svaka popularna aplikacija za bilješke pohranjuje podatke na udaljenim poslužiteljima. Jedan propust, jedan zahtjev vlasti, jedna promjena pravila — i vaše privatne misli su otkrivene. Format .note to čini nemogućim po svojoj prirodi.
📦
Što je unutar .note datoteke
Datoteka .note je samostalni JSON paket: sadržaj bilješke, metapodaci, oznake i — pri uključenoj enkripciji — AES-256 šifrirani tekst koji samo vaša lozinka može dešifrirati. Bez vlasničkog formata, bez vezivanja za dobavljača.
🌐
Offline-first, uvijek prenosiv
Izvezite bilo koju bilješku kao datoteku .note i nosite je na USB-u, šaljite putem Signala ili arhivirajte na desetljeća. Uvezite natrag u Local Notes na bilo kojem uređaju — bez računa, bez interneta.
🛡️
Zero-knowledge kao temelj
Pri šifriranju bilješke ključ nikada ne napušta vaš uređaj. Local Notes koristi ugrađeni Web Crypto API preglednika — isti standard koji koriste banke. Čak i ako netko presretne vašu datoteku .note, vidjet će samo šifrirani tekst.

FAQ

Česta pitanja

Sve što ste željeli znati prije početka.

Da, potpuno. Local Notes je open source softver pod MIT licencom. Bez plaćenih planova, bez skrivenih naknada — zauvijek. Sve značajke su dostupne besplatno.

Vaše bilješke se pohranjuju u IndexedDB vašeg preglednika — lokalnoj bazi podataka ugrađenoj u svaki moderni preglednik. Ništa se nikada ne šalje na poslužitelj.

Brisanje pohrane preglednika će obrisati vaše bilješke. Preporučujemo redovito izvoziti bilješke kao .note, HTML ili Markdown datoteke.

Kada se enkripcija uključi, bilješke se šifriraju pomoću Web Crypto API ugrađenog u preglednik. Ključ izveden iz vaše lozinke šifrira svaku bilješku prije pohrane.

Local Notes je namjerno samo lokalan — bez sinkronizacije. Za prijenos bilješki između uređaja izvezite ih kao .note datoteke i uvezite na drugom uređaju.

Da. Local Notes je potpuno otvoren pod MIT licencom. Možete pregledati, forkati i doprinijeti na GitHub.

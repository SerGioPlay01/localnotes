Příběh vzniku

Proč vznikl Local Notes

Vše začalo prostou frustrací — najít aplikaci pro poznámky, která je skutečně soukromá, rychlá a zdarma.

Problémy stávajících aplikací
☁️
Vše se synchronizuje se serverem. Notion, Evernote, Google Keep — vaše poznámky žijí na cizím počítači. Jeden únik a data jsou odhalena.
💳
Šifrování stojí peníze. Většina aplikací skrývá skutečné šifrování za placeným plánem. Soukromí by nemělo být prémiovou funkcí.
📋
Registrace všude. Abyste napsali poznámku, potřebujete email, heslo a souhlas se zásadami, které umožňují používat vaše data.
🌐
Žádná podpora offline. Většina webových aplikací je bez internetu k ničemu. Vaše poznámky by měly být dostupné vždy a všude.
Začátek hledání
Hledání soukromé aplikace pro poznámky
Vyzkoušeny desítky aplikací. Každá vyžadovala účet, synchronizovala se s cloudem nebo účtovala za základní funkce. Nic se nezdálo skutečně soukromé.
Uvědomění
Prohlížeč už má vše potřebné
Moderní prohlížeče mají IndexedDB pro lokální úložiště, Web Crypto API pro šifrování AES-256 a Service Workers pro offline provoz. Server není potřeba.
První prototyp
Vytvořena minimální funkční verze
Jednoduchý editor, lokální úložiště, základní šifrování. Žádné frameworky, žádné závislosti. Jen HTML, CSS a čistý JavaScript.
Rozvoj nápadu
Štítky, kalendář, 12 jazyků, PWA
Přidáno formátování textu, barevné štítky, vestavěný kalendář, vyhledávání s transliterací a plná lokalizace do 12 jazyků. Zabaleno jako PWA.

Formát .note

Vaše data, váš formát

Soubor .note je přenosný šifrovaný kontejner navržený s jediným cílem: vaše poznámky patří vám — ne platformě, cloudu ani firmě.

🔒
Zrozený z problému soukromí
Každá populární aplikace pro poznámky ukládá data na vzdálených serverech. Jeden únik, jedna žádost úřadů, jedna změna zásad — a vaše soukromé myšlenky jsou odhaleny. Formát .note to znemožňuje svou podstatou.
📦
Co je uvnitř souboru .note
Soubor .note je samostatný JSON balíček: obsah poznámky, metadata, štítky a — při zapnutém šifrování — šifrovaný text AES-256, který může dešifrovat pouze vaše heslo. Žádný proprietární formát, žádné uzamčení k dodavateli.
🌐
Offline-first, vždy přenositelný
Exportujte libovolnou poznámku jako soubor .note a noste ji na USB, posílejte přes Signal nebo archivujte na desetiletí. Importujte zpět do Local Notes na jakémkoli zařízení — bez účtu, bez internetu.
🛡️
Zero-knowledge jako základ
Při šifrování poznámky klíč nikdy neopustí vaše zařízení. Local Notes používá vestavěné Web Crypto API prohlížeče — stejný standard, který používají banky. I kdyby někdo zachytil váš soubor .note, uvidí pouze šifrovaný text.

FAQ

Časté otázky

Vše, co jste chtěli vědět před začátkem.

Ano, zcela. Local Notes je open source software pod licencí MIT. Žádné placené plány, žádné skryté poplatky — navždy. Všechny funkce jsou dostupné zdarma.

Vaše poznámky jsou uloženy v IndexedDB vašeho prohlížeče — lokální databázi zabudované v každém moderním prohlížeči. Nic se nikdy neodesílá na server.

Vyčištění úložiště prohlížeče smaže vaše poznámky. Doporučujeme pravidelně exportovat poznámky jako .note, HTML nebo Markdown soubory.

Při zapnutí šifrování jsou poznámky šifrovány pomocí Web Crypto API zabudovaného v prohlížeči. Klíč odvozený z vašeho hesla šifruje každou poznámku před uložením.

Local Notes je záměrně pouze lokální — bez synchronizace. Pro přenos poznámek mezi zařízeními je exportujte jako .note soubory a importujte na druhém zařízení.

Ano. Local Notes je plně otevřený pod licencí MIT. Můžete prozkoumat, forkovat a přispívat na GitHub.

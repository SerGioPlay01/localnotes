2. Описание сервиса
Local Notes — бесплатное браузерное приложение для создания заметок. Все заметки хранятся исключительно в браузере пользователя с использованием LocalStorage и/или IndexedDB и шифруются алгоритмом AES-256. Приложение не имеет серверного хранилища, учётных записей и не требует регистрации.

10. Офлайн и онлайн режимы
Приложение поддерживает два режима работы сети:
— Онлайн режим: приложение работает в штатном режиме с полным доступом ко всем функциям. Google Analytics может собирать обезличенную статистику использования при наличии подключения к интернету.
— Офлайн режим: приложение продолжает полноценно работать, используя кэшированные данные в браузере. Все заметки остаются доступными для просмотра и редактирования. В офлайн режиме данные не передаются ни на какие серверы. При потере подключения к интернету приложение отображает баннер-уведомление.
Вы можете вручную переключаться между режимами с помощью переключателя сети в нижней части приложения. Выбранный режим сохраняется в localStorage браузера.

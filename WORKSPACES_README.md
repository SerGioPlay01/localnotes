# Рабочие пространства (Workspaces)

## Описание

Добавлена новая функциональность **рабочих пространств** для организации заметок по разным вкладкам/категориям.

## Возможности

### 1. Вкладки рабочих пространств
- Вкладки отображаются под хедером приложения
- Каждое пространство имеет свой цвет для визуального различия
- Активное пространство подсвечивается
- Звёздочка (★) отмечает пространство по умолчанию

### 2. Управление пространствами
- **Создание**: Кнопка "+" для создания нового пространства
- **Переименование**: Через меню вкладки (⋮)
- **Удаление**: Через меню вкладки (⋮)
- **Установка по умолчанию**: Через меню вкладки (⋮)

### 3. Сохранение заметок
При создании новой заметки перед сохранением появляется диалог выбора рабочего пространства:
- Список всех доступных пространств
- Текущее пространство отмечено
- Возможность создать новое пространство прямо из диалога
- Кнопка отмены для возврата к редактированию

### 4. Фильтрация заметок
- Заметки автоматически фильтруются по выбранному пространству
- При переключении между пространствами отображаются только заметки этого пространства
- Поиск работает только в рамках текущего пространства

## Технические детали

### Файлы
- `js/workspaces.js` - Основной класс WorkspacesManager
- `js/workspaces-integration.js` - Интеграция с существующим кодом
- `js/workspaces-translations.js` - Переводы на 12 языков
- `css/workspaces.css` - Стили для UI рабочих пространств

### Хранение данных
- Список пространств хранится в `localStorage` под ключом `workspaces`
- Каждая заметка имеет поле `workspaceId` для привязки к пространству
- Заметки без `workspaceId` автоматически привязываются к пространству по умолчанию

### Структура данных пространства
```javascript
{
    id: 'ws_timestamp_random',
    name: 'Название пространства',
    isDefault: false,
    createdAt: timestamp,
    color: '#4CAF50'
}
```

## Адаптивность

- **Десктоп**: Полноценные вкладки с названиями
- **Планшет**: Компактные вкладки с прокруткой
- **Мобильные**: Оптимизированный UI с сенсорным управлением

## Безопасность

- Все данные хранятся локально в браузере
- Шифрование заметок работает независимо от пространств
- При удалении пространства все его заметки также удаляются

## Использование

1. **Создание пространства**: Нажмите кнопку "+" справа от вкладок
2. **Переключение**: Кликните на нужную вкладку
3. **Создание заметки**: При сохранении выберите пространство
4. **Управление**: Нажмите "⋮" на вкладке для дополнительных опций

## Примечания

- Нельзя удалить пространство по умолчанию (сначала нужно назначить другое)
- При первом запуске автоматически создаётся пространство "Default"
- Существующие заметки автоматически привязываются к пространству по умолчанию

Історія створення

Чому з'явився Local Notes

Все почалося з простого розчарування — знайти застосунок для нотаток, який справді приватний, швидкий і безкоштовний.

Проблеми існуючих застосунків
☁️
Все синхронізується з сервером. Notion, Evernote, Google Keep — ваші нотатки зберігаються на чужому комп'ютері. Один витік — і дані розкриті.
💳
Шифрування коштує грошей. Більшість застосунків ховають справжнє шифрування за платною підпискою. Приватність не повинна бути преміум-функцією.
📋
Скрізь потрібна реєстрація. Щоб написати нотатку, потрібен email, пароль і згода з політикою, що дозволяє використовувати ваші дані.
🌐
Немає офлайн-підтримки. Більшість веб-застосунків марні без інтернету. Ваші нотатки мають бути доступні завжди і скрізь.
Початок пошуку
Пошук приватного застосунку для нотаток
Перепробувано десятки застосунків. Кожен вимагав акаунт, синхронізувався з хмарою або брав гроші за базові функції. Ніщо не здавалося справді приватним.
Усвідомлення
Браузер вже має все необхідне
Сучасні браузери мають IndexedDB для локального зберігання, Web Crypto API для шифрування AES-256 і Service Workers для офлайн-роботи. Сервер не потрібен.
Перший прототип
Створено мінімальний робочий варіант
Простий редактор, локальне сховище, базове шифрування. Жодних фреймворків, жодних залежностей. Тільки HTML, CSS і чистий JavaScript.
Розвиток ідеї
Теги, календар, 12 мов, PWA
Додано форматований текст, кольорові теги, вбудований календар, пошук з транслітерацією та повна локалізація на 12 мов. Упаковано як PWA для нативного встановлення.
Сьогодні — v1.1.0
Відкритий код і безкоштовно для всіх
Local Notes поширюється під ліцензією MIT і буде безкоштовним завжди. Мета проста: дати кожному безпечне приватне місце для записів.
🔒
Відповідь: зберігати все в браузері
Local Notes використовує ваш браузер як захищене сховище. Шифрування AES-256, IndexedDB, Service Worker — все вбудовано в сучасні браузери. Без сервера, без акаунту, без компромісів.

Формат .note

Ваші дані, ваш формат

Файл .note — це портативний зашифрований контейнер, створений з однією метою: ваші нотатки належать вам, а не платформі, хмарі чи компанії.

🔒
Народжений з проблеми приватності
Кожен популярний застосунок для нотаток зберігає дані на віддалених серверах. Один витік, один запит влади, одна зміна політики — і ваші особисті думки розкриті. Формат .note робить це неможливим за своєю природою.
📦
Що всередині файлу .note
Файл .note — це самодостатній JSON-пакет: вміст нотатки, метадані, теги і — при увімкненому шифруванні — шифртекст AES-256, який може розшифрувати лише ваш пароль. Жодного пропрієтарного формату, жодної прив'язки до вендора.
🌐
Офлайн-перший, завжди переносимий
Експортуйте будь-яку нотатку як файл .note і носіть на флешці, надсилайте через Signal або архівуйте на десятиліття. Імпортуйте назад у Local Notes на будь-якому пристрої — без акаунту, без інтернету.
🛡️
Zero-knowledge за замовчуванням
При шифруванні нотатки ключ ніколи не залишає ваш пристрій. Local Notes використовує вбудований у браузер Web Crypto API — той самий стандарт, що використовують банки. Навіть якщо хтось перехопить ваш файл .note, він побачить лише шифртекст.

FAQ

Часті запитання

Все, що ви хотіли знати перед початком.

Так, повністю. Local Notes — програмне забезпечення з відкритим кодом під ліцензією MIT. Жодних платних планів, жодних прихованих платежів — назавжди. Всі функції доступні всім безкоштовно.

Ваші нотатки зберігаються в IndexedDB вашого браузера — локальній базі даних, вбудованій у кожен сучасний браузер. Нічого ніколи не надсилається на сервер. Ваші дані не залишають пристрій.

Очищення сховища браузера видалить ваші нотатки. Рекомендуємо регулярно експортувати нотатки у форматах .note, HTML або Markdown. Файли .note можна імпортувати назад у будь-який час.

При увімкненні шифрування нотатки шифруються за допомогою Web Crypto API, вбудованого у браузер. Ключ, похідний від вашого пароля, використовується для шифрування кожної нотатки перед збереженням. Без правильного пароля дані нечитабельні.

Local Notes навмисно працює лише локально — синхронізації немає за задумом. Для перенесення нотаток між пристроями експортуйте їх у файли .note та імпортуйте на іншому пристрої.

Так. Local Notes повністю відкритий під ліцензією MIT. Ви можете вивчити, форкнути та зробити внесок на GitHub. Жодних чорних ящиків.

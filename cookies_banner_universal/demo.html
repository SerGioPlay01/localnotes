Local Notes Cookies Information

This banner provides detailed information about the specific cookies used in Local Notes:

üîß Necessary Cookies

  • localnotes_notes_data - Encrypted notes storage
  • localnotes_encryption_key - AES-256 encryption keys
  • localnotes_theme - Dark/light theme preference
  • preferredLanguage - User language preference
  • localnotes_view_mode - Grid/list view preference
  • localnotes_pwa_install - PWA installation status
  • localnotes_session - Session management

üìä Analytics Cookies

  • _ga - Google Analytics user identification
  • _ga_* - Google Analytics 4 measurement
  • _gid - Google Analytics session data
  • _gat - Google Analytics throttling
  • G-HR9HLBQFCR - Local Notes tracking ID

üéØ Marketing Cookies

Currently not used in Local Notes. Reserved for future features like social sharing or promotional content.

Integration Instructions

To integrate this cookies banner into your Local Notes pages:

  1. Copy the cookies-banner.js file to your project
  2. Include it in your HTML pages: <script src="cookies_banner_universal/cookies-banner.js"></script>
  3. The banner will automatically initialize and detect the current language
  4. Use the public API for advanced control: CookiesBanner.init(), CookiesBanner.hasConsent(), etc.
  5. Google Analytics will be automatically managed based on user consent
